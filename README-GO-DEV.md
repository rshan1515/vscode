# Go Development & Debugging in VS Code Dev Container\n\n## Quick Start\n\n1. **Open your project in VS Code.**\n2. **Add the files below to your repository.**\n3. **Rebuild your Dev Container:**  \n   - Press `F1` → `Dev Containers: Rebuild Container`\n4. **Open `main.go`, set breakpoints, and press `F5` to debug!**\n\n## Files Included\n\n- `.devcontainer/devcontainer.json` — Adds Go support.\n- `.devcontainer/Dockerfile` — Installs Go (if you want Dockerfile-based setup).\n- `.vscode/launch.json` — Debug configuration for Go.\n- `main.go` — Sample Go code for debugging.\n\n## Troubleshooting\n\n- Make sure Delve (`dlv`) is installed (`go install github.com/go-delve/delve/cmd/dlv@latest`).\n- If breakpoints don’t hit, use `"mode": "debug"` in your launch config.\n- Permissions: On Linux/Mac, ensure VS Code and dlv have permission to debug (run as admin if needed).
